# Semantic Video Summarization Pipeline - FE

## About project

This sub-folder is front-end part of [SVSP](../README.md)

## How to run

1. install package
```PowerShell
npm install
```

2. Run local server
```PowerShell
npm run dev
```

3. Open in webbrowser  
[http://localhost:5173](http://localhost:5173/)
